# 河南方言语音板 · 桌面应用 发布说明

> 纯原生桌面软件（PySide6 / Qt），**不再依赖浏览器**，双击即用。

## 一、这是什么
一个帮你「说河南话、懂普通话」的桌面小工具：输入或录一句河南方言，立刻得到
**普通话翻译 + 汉语拼音（带声调）+ 逐词拼音（跟读）+ 释义**。还附带：
- 🗺 **方言地图**：点河南 18 个地市，看各地说法差异
- 🎮 **猜方言**：听/看一句方言，选出正确意思，计分闯关
- 🖼 **方言画廊**：AI 乡土插画卡片，可保存
- 📜 **我的记录**：历史自动保存、好句手动收藏（数据只存本机，不上传）
- 🔊 **语音朗读**：一键朗读普通话或原句（在线 Edge TTS 真人发音，离线自动降级系统语音）
- 🪄 **智能纠错**：录音识别结果接近词库词条时自动纠正（如「弄啥来」→「弄啥嘞」）
- 🎨 **界面打磨**：启动 Splash、页面淡入动画、🔍 长辈模式（放大字号、增强对比）

## 二、怎么用（发布版）
1. 推荐直接发 `河南方言语音板_vYYYYMMDD.zip`，对方解压即得 `河南方言语音板/` 文件夹（内含 exe 与依赖）；若从 `dist/` 直接拷贝，请拷 `河南方言语音板_build/` 整个文件夹，并改名为 `河南方言语音板`。
2. 进入文件夹，**双击 `河南方言语音板.exe`** 即可运行，会弹出软件窗口，无需安装 Python、无需开浏览器。文件夹里的 `_internal` 等是程序依赖，**必须和 exe 放在一起**，不要单独挪动 exe。
3. 在「语音板」输入或选一句示例（如「弄啥嘞」），点「生成普通话 + 拼音 + 释义」；点「▶ 朗读」可听普通话或原句发音。
4. 想用麦克风：点「🎙 录音识别」（需系统有麦克风；识别默认走演示模式词库，配置密钥后可真实识别）。
5. 解析过的会自动进「我的记录 → 历史」；点结果区的「⭐ 收藏」进收藏。

> **性能**：发布为**文件夹版（onedir）**，启动无需解压、秒级开窗，不再有旧单文件版（onefile）的 273MB 解压白屏。多轮运行时提速——① 朗读（Edge TTS）结果按「音色+文本」**磁盘缓存**，同一句话反复听秒开、省流量、可离线复用，且不再产生临时文件泄漏；② 方言解析结果**内存缓存**（LRU），重复解析不重复计算；③ 启动时预计算全部词库拼音，运行时零 pypinyin 调用；④ `edge_tts` 改为**首次朗读时才懒加载**，把较重的 aiohttp 依赖移出启动关键路径，主窗口弹出更快；⑤ 演示贪婪匹配**按首字建索引**，匹配时只扫当前字对应桶，不再每字符全量扫描全部词条；⑥ 智能纠错用「词条集合 O(1) 命中 + 按长度分桶」，避免全量 Levenshtein 扫描。

**未配置任何密钥时，自动进入「演示模式」**：离线、不联网，用内置 735+ 条方言词库（多包聚合，含 18 地市 region 对比数据）+ pypinyin 兜底，全链路可演示。

## 三、想用真实语音识别 / 大模型释义（可选）
1. 在同目录放一个 `.env` 文件（可复制 `.env.example` 改名），填入：
   - `XF_APPID` / `XF_API_KEY` / `XF_API_SECRET`：讯飞「语音听写」方言 ASR（河南话）
   - `LLM_API_KEY` + 可选 `LLM_BASE_URL` / `LLM_MODEL`：OpenAI 兼容大模型做释义
2. **首次真实识别需要 FFmpeg**（把录音转成 16k 单声道 PCM）：
   - Windows 装 [FFmpeg](https://ffmpeg.org/) 并加入 PATH，或把 `ffmpeg.exe` 放在 exe 同目录。
3. 重新双击 exe 即切换为「云端模式」。无麦克风/无密钥时，仍可手动输入文字解析。

> 💡 **更简单的方式**：应用内「⚙ 设置」面板已提供「大模型 API Key / Base URL / 模型名」三项，填写后保存即**即时生效并自动写入本机 `.env`**，无需手动编辑配置文件、无需重启。支持 DeepSeek、通义千问（DashScope 兼容模式）、本地 Ollama（Base URL 填 `http://localhost:11434/v1`）。

> 想彻底离线、隐私零上传：装 `pip install funasr`，程序会自动走本地 FunASR 识别。

## 四、功能与隐私说明
- 历史/收藏存于本机用户目录 `~/.henan_dialect/user_data.json`，**不会上传任何服务器**。
- 演示模式完全离线；仅当配置了讯飞/大模型密钥并发起识别/解析时，才会联网调用对应服务。
- 释义由 AI 或内置词库生成，**仅供参考**。

## 五、自己重新打包（开发者）
环境（建议虚拟环境）安装依赖：
```
pip install -r requirements.txt
pip install pyinstaller
```
双击 `build_exe.bat`，或手动执行：
```
pyinstaller --noconsole --onedir --name "河南方言语音板_build" ^
  --add-data "dialect_dict.json;." --add-data "assets;assets" ^
  --collect-all PySide6 --collect-all edge_tts --collect-data pypinyin ^
  --exclude-module PySide6.Qt3DAnimation ...（其余裁剪见 河南方言语音板.spec） main.py
```
产物在 `dist/河南方言语音板_build/`（文件夹版）。正式名 `河南方言语音板/` 偶尔会被 Defender/索引服务持锁导致无法覆盖；更稳妥的做法是用环境变量指定**全新时间戳目录**彻底避开清锁 `PermissionError`：`set HB_BUILD_DIR=河南方言语音板_b<时间戳>`（注意必须 `export`/`set` 导出给子进程，否则 spec 读不到会回退默认名）再 `pyinstaller`。`make_dist.py` 会自动选取 `dist/` 下最新、含 exe 的构建目录作为打包源，zip 内顶层统一为 `河南方言语音板/`。

## 六、目录结构（源码）
```
main.py              桌面入口（QApplication + 主窗口 + 导航）
backend.py           后端逻辑：词库 / 拼音 / 讯飞ASR / FunASR / 大模型（无 Flask）
recorder.py          麦克风录音（sounddevice）+ 实时音量电平
store.py             历史/收藏 JSON 持久化（用户目录）
theme.py             暗灰+翠绿「情报终端」QSS 主题
page_main.py        语音板页
page_map.py         方言地图页
page_game.py        猜方言游戏页
page_gallery.py     方言画廊页
page_records.py     我的记录页
dialect_dict.json   主词库 + dialect_dict_extra.json 扩展包（共 735+ 条，含 18 地市 region 数据，支持多包聚合与用户自定义导入）
assets/illustrations/   4 张 AI 插画
```

## 七、安装包与分发包（方便分发）
- **一键分发包**：运行 `python make_dist.py`，它会自动选取 `dist/` 下最新、含 exe 的构建目录（对外统一命名为 `河南方言语音板/`） + 文档 + `.env.example` 压成 `河南方言语音板_vYYYYMMDD.zip`（约 501MB，解压后约 980MB，体积主要来自 Qt6 运行时 + 238MB 离线识别模型），直接发给他人解压即用。
- **安装向导（setup.exe）**：在已装 [InnoSetup 6](https://jrsoftware.org/isdl.php) 的本机，双击运行项目根目录的 `build_installer.bat`（自动挑选 `dist/河南方言语音板_b*` 最新构建目录并编译），得到 `dist/河南方言语音板_setup.exe`；或手动用 InnoSetup 打开 `installer/河南方言语音板_installer.iss` 按 F9 编译。安装包支持安装到当前用户 Program Files、创建桌面/开始菜单快捷方式、标准卸载，无需管理员权限。
- **更新**：重新打包后，把新 `dist/河南方言语音板.exe` 覆盖原文件即可；分发包按日期命名便于区分版本。
